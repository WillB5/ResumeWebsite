Yu-Gi-Oh! Card Look up
This is an Android application I built to look up cards from the trading card game called "Yu-Gi-Oh!" It was designed to look up cards by name or ID number, since cards can be printed in different languages, but will always have the same ID number. This is useful if you are in a tournament setting and would like to check a card description of another player's card but do not understand the language it is printed in. The app will also display any restrictions placed on the card for competitive play if you need to make sure your cards are legal in a tournament.
Built using: Android Studio, Kotlin, Google's Volley Library, and an API provided by ygoprodeck.com
